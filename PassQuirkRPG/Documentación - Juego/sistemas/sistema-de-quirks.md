# ✨ Sistema de Quirks

#### Sistema de Poder de Quirks (Básico)

| Concepto       | Descripción                                                                                  |
| -------------- | -------------------------------------------------------------------------------------------- |
| Poder Base     | Cada Quirk tiene un poder base numérico que afecta daño, defensa o buff según su tipo.       |
| Escalado       | El poder crece con el nivel del personaje y mejoras del Quirk (entrenamiento, equipamiento). |
| Bonificaciones | Los PassQuirks añaden multiplicadores temporales o permanentes a todas las habilidades.      |
| Rareza         | Rareza (Común, Raro, Legendario) define el rango inicial del poder y la velocidad de subida. |
