# ⏲️ Sistema Tiempo

#### Sistema de Tiempo

| Funcionalidad        | Descripción                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------ |
| Tiempo Real / Ciclos | El juego tendrá un ciclo día/noche con duración configurable (ej: 24 minutos = 24 horas reales). |
| Eventos Programados  | Eventos y zonas especiales aparecerán en determinados momentos del día o durante días concretos. |
| Reloj Interno        | Controla la aparición de enemigos, drops raros y la activación de misiones especiales.           |
| Tiempo en Mundo      | Tiempo afecta la exploración, con ciertas habilidades o Quirks que modifican duración o efectos. |
